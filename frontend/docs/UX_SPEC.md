# Ojaa Marketplace - UX Specification Document

## Overview

This document provides detailed page-by-page UX wireframe descriptions and user flows for the Ojaa Marketplace, a Shein-style African fashion e-commerce platform with integrated virtual try-on capabilities.

---

## Design System Foundation

### Visual Language
- **Primary Color**: Warm Ember (#E85A2C) - Afrocentric warmth
- **Accent Color**: Sand Gold (#FBF5E8) - Premium feel
- **Typography**: Inter (clean, modern, high readability)
- **Spacing**: 8px base unit grid system
- **Border Radius**: 8px (cards), 4px (buttons), 24px (pills)
- **Shadows**: Subtle elevation (0 2px 8px rgba(0,0,0,0.08))

### Responsive Breakpoints
- Mobile: 320px - 767px
- Tablet: 768px - 1023px
- Desktop: 1024px - 1399px
- Large Desktop: 1400px+

## 12. ACCESSIBILITY CONSIDERATIONS

- **Keyboard Navigation**: All interactive elements focusable via Tab
- **Screen Readers**: ARIA labels on icons and interactive elements
- **Color Contrast**: WCAG 2.1 AA compliant (4.5:1 minimum)
- **Touch Targets**: Minimum 44x44px for mobile
- **Focus Indicators**: Visible focus rings on all elements
- **Alt Text**: Descriptive alt text for all product images
- **Motion**: Respect `prefers-reduced-motion` for animations
- **Virtual Try-On**: Text descriptions of try-on results for accessibility
